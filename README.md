# Canvas2026
“Canvas 2026” is a modern, aesthetic vision-board style website. It represents goals, clarity, ambition, and creativity for the year 2026.
The HTML builds the full layout of the website.
It includes: Navigation bar, Hero section, Image mosaic, Vision cards, Feature strip, Quote section, and Footer.
Each section is divided using semantic tags like <nav>, <section>, <div>, and <footer> for proper structure.
CSS is written inside the <style> tag (Internal CSS).
It controls colors, fonts, spacing, alignment, animations, hover effects, and responsiveness.
CSS variables (:root) are used for consistent color themes.
	•	Gradient backgrounds and soft glow effects for aesthetic feel.
	•	Hover animations on images and buttons.
	•	Keyframe animations (fadeUp, fadeIn, drift) for smooth entrance effects.
	•	Noise overlay and blur glow elements for a premium look.
